# Crosshair0

عربي:

Crosshair0 — أوفرلي كروسير احترافي لألعاب الكمبيوتر

أوفرلي شفاف خفيف يشتغل فوق أي لعبة. لا يحتاج حقن ولا عليه حظر. يدعم CS2, Valorant, FiveM وكل الألعاب (ملء الشاشة أو بدون إطار).

الميزات:

6 أنواع كروسير: صليب، نقطة، دائرة، شكل X، شكل T، مع إمكانية طبقة ثانية
وضع RGB كامل مع سرعة قابلة للتعديل + دوران + إزاحة X/Y
التبديل التلقائي حسب اللعبة — يكشف اللعبة ويحمل البروفايل المخصص لها
معرض إعدادات مسبقة مع مفضلة — حفظ، تحميل، مشاركة عبر رابط
مدير بروفايلات مع إعادة تسمية، استيراد/تصدير، تراجع/إعادة (Ctrl+Z/Y)
تصدير PNG + نسخ/لصق عبر الحافظة
مولد كود CS2 و Valorant
دعم الشاشات المتعددة
واجهة عربية/إنجليزية مع دعم RTL
خفيف جدًا (~5 MB RAM) — يعمل من علبة النظام
الألعاب: CS2, Valorant, Apex Legends, Fortnite, Rainbow Six, Warzone, FiveM، وأي لعبة أخرى.


English:

Crosshair0 — Professional Crosshair Overlay for PC Games

A lightweight, customizable crosshair overlay that works on top of any game. No injection, no bans. Supports CS2, Valorant, FiveM, and all windowed/fullscreen games.

Features:

6 crosshair types: Cross, Dot, Circle, X-Shape, T-Shape, with optional second layer
Full RGB mode with adjustable speed + rotation + offset X/Y
Per-game auto-switch — detects your game and loads the assigned profile
Custom presets gallery with favorites — save, load, share as link
Profile manager with rename, import/export, undo/redo (Ctrl+Z/Y)
Canvas PNG export + clipboard copy/paste
CS2 console code + Valorant code generator
Multi-monitor support
RTL Arabic/English interface
Lightweight (~5 MB RAM) — runs in system tray
Games: CS2, Valorant, Apex Legends, Fortnite, Rainbow Six, Warzone, FiveM, and any other game.
