## English

**Crosshair0** — A lightweight, customizable crosshair overlay for PC games. Works on top of any game. No injection, no bans.

### Features

| Feature | Description |
|---------|-------------|
| **6 Types** | Cross, Dot, Circle, X-Shape, T-Shape, with independent second layer |
| **Dynamic RGB** | Full RGB mode with adjustable speed |
| **Rotation + Offset** | Rotate the crosshair at any angle + real-time X/Y offset |
| **Auto-Switch** | Detects running games and loads the assigned profile automatically |
| **Preset Gallery** | Custom presets with favorites, save/share via link |
| **Profile Manager** | Save, load, rename, import/export, undo/redo |
| **PNG Export** | Export crosshair as an image |
| **Code Generator** | Generates CS2 and Valorant console code |
| **Multi-Monitor** | Pick any display for the overlay |
| **Lightweight** | ~5 MB RAM, runs in system tray |
| **RTL Support** | Full Arabic/English interface |
| **Auto-Update** | Updates download and install automatically |

### Installation

1. Download the latest release from the [Releases page](https://github.com/0xxsrp/Crosshair0/releases/latest)
2. Run the installer (`Crosshair0 Setup X.X.X.exe`)
3. The app starts automatically after installation
4. The overlay appears on top of all games

> Future versions auto-download and install — you only need to install manually once.
